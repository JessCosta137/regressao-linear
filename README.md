# Regressão Linear - PrograMaria

Projeto desenvolvido durante o curso **Eu Progr{amo}** da PrograMaria.

## 📌 Sobre o projeto
Análise e modelagem de dados de salários utilizando Regressão Linear, com foco em identificar os fatores que influenciam a remuneração de profissionais.

## 🛠️ Tecnologias utilizadas
- Python
- Pandas
- Scikit-learn
- Google Colab

## 📊 Etapas do projeto
- Carregamento e exploração dos dados
- Feature Engineering (get_dummies, fillna, StandardScaler)
- Divisão treino/teste com train_test_split
- Modelagem com Regressão Linear
